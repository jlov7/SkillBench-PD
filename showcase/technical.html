Technical overview

Benchmark, orchestrate, gate, and report.

SkillBench-PD combines deterministic task fixtures, matrix orchestration, and statistical regression checks to provide actionable release signals.

Core architecture

tasks + skills --> prompt modes (baseline|naive|progressive) --> provider inference --> rule/llm judge --> result records --> report outputs + regression gate

Orchestration Engine

`bench/experiment.py` expands matrix cases, executes in parallel, supports retries, QPS limits, and checkpoint resume.

Regression Gate

`bench/regression.py` computes bootstrap confidence intervals, permutation p-values, effect sizes, and PASS/FAIL verdicts.

Shareable Reports

`bench/report.py` emits CSV/Markdown/HTML artifacts that are portable across static hosting.

Reference command (full path)

uv run skillbench-pd \ --orchestrate \ --modes baseline naive progressive \ --tasks tasks/t1_rewrite_brand.json tasks/t2_format_policy.json \ --repetitions 3 \ --matrix-models mock-a mock-b \ --matrix-judges rule \ --max-workers 4 \ --retry-attempts 2 \ --rate-limit-qps 5 \ --checkpoint-path results/checkpoint.jsonl \ --fail-on-regression
