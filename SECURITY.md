# Security Policy

## Scope
SkillBench-PD is a personal independent research project and is provided without production support guarantees.

## Supported Versions
Security fixes are applied on the latest `master` branch.

## Reporting a Vulnerability
Please report suspected vulnerabilities privately via GitHub Security Advisories:
- [Report a vulnerability](https://github.com/jlov7/SkillBench-PD/security/advisories/new)

Do not open public issues for active vulnerabilities.

## Secure Usage Notes
- Never commit API keys or credentials.
- Use synthetic or non-sensitive benchmark data.
- Review generated reports before sharing externally.
